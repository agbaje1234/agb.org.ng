LINE BREAK IN HTML :
If you use the <.br> tag , anything following it starts from the next line .
This tag is an example of an empty element, there is no need for opening and closing tags, as there is nothing to go in between them.
CENTERING CONTENT :
To center any content in HTML You can use <.center.> tag this will to put any content inside the tag in the center of the page or any table cell. Example for Centering content :
REMOVE THE POINTS IN THE TAGS:

<.head.>
<.title.>Centering content<./title.>
<./head.>
<.body.>
<.p.>This is not in the center .<./p.>
<.center.>

<./center.>
<./body.>
<./html.>

HORIZONTAL LINE :
To create a horizontal line between a section of document the <.hr> tag can be used it creates a line from the current position in the document to the right margin and breaks the line accordingly .
